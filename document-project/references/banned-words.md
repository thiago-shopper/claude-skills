# Banned words

Words and phrases to strip during Step 8 (voice pass). They add no information and signal template prose. Match case-insensitively. When you find one, either delete the word or rewrite the sentence into a concrete claim.

## Marketing adjectives

- seamless
- seamlessly
- robust
- powerful
- enterprise-grade
- world-class
- cutting-edge
- state-of-the-art
- blazing-fast
- blazing fast
- lightning-fast
- best-in-class
- next-generation
- next-gen
- revolutionary
- modern
- elegant
- beautiful
- delightful
- magical

## Effort-minimizing adverbs

- simply
- easily
- effortlessly
- intuitively
- conveniently
- automatically *(only banned when used as filler — keep it when describing a real automated behavior)*

## Vague scope claims

- comprehensive
- complete
- full-featured
- extensive
- holistic
- end-to-end *(unless describing an actual e2e test pipeline)*
- one-stop
- all-in-one

## Verbs that hide what the thing does

- leverage / leverages / leveraging
- utilize / utilizes / utilizing
- unlock / unlocks
- empower / empowers / empowering
- streamline / streamlines / streamlining
- facilitate / facilitates / facilitating
- enable / enables *(banned when used as filler — keep it when describing a real enablement)*
- harness / harnesses

## Hedge phrases

- aims to
- is designed to
- is intended to
- helps to
- allows you to
- makes it easy to
- makes it possible to
- provides the ability to
- gives you the power to

## Filler openings

- This project is a
- This repository is a
- This is a
- Welcome to
- In this repository you'll find
- This README will guide you through
- This document describes
- The purpose of this project is

## Filler closings

- Made with love
- Built with ❤️
- Built with passion
- Happy coding
- We hope you enjoy

## Catch-all

Any sentence that, when removed, would not reduce the reader's understanding of how to use, run, test, or deploy the project — delete it.
