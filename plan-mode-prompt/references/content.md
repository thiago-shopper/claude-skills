# content template

## When to use
Recurring content product: newsletter, podcast, blog, YouTube channel, Substack series, book chapters shipped over time. Anything with a **cadence**.

## Default audience
A specific niche the user identifies in the draft (or that the topic implies). Default channel mix: the topic's natural-fit platform + 1 cross-promo channel (e.g. newsletter + Twitter; podcast + LinkedIn).

## Default timeline + milestones
- Horizon: **12 weeks** to a first sustained ramp.
- Cadence: **weekly** by default (revise for podcasts if biweekly is healthier).
- Milestones:
  1. Week 1 — landing page + positioning / manifesto live.
  2. Week 2 — first issue / episode ships; cadence locked.
  3. Week 6 — first cross-promo or guest swap with a peer creator; intermediate audience checkpoint.
  4. Week 12 — review against initial targets; decide whether to keep cadence, increase, or pivot.

## Default resources
- Solo creator.
- ≤ 5 hours per week of production capacity.
- Free-tier platform (Substack / Beehiiv / Anchor / similar).
- Existing personal social presence as the day-zero distribution.

## Likely risks (template-derived)
- Cadence slip in weeks 3–5 (the post-launch trough) → mitigation: bank 3 drafts / episodes before week 1 launches.
- Topic drift away from the niche → mitigation: collect 3 reader/listener replies each week and mine them for next issue's hook.
- Audience growth flatlines after the initial network bump → mitigation: schedule one cross-promo / guest by week 6 at the latest.
- Production fatigue / burnout → mitigation: define what "skip a week" looks like and pre-write the "I'm taking a week off" message you'll send.
- Slow start anxiety → mitigation: set an absolute number you will accept at week 4 (e.g. 50 subs) and commit publicly to keep shipping past it.

## Default success criteria
- Leading metric: ≥ 60% open / completion rate on the first 8 issues / episodes (a healthy engagement signal regardless of size).
- Lagging metric: a numeric audience target by week 12 — set based on draft if given, else propose 1000 subs / 500 listeners and label as Assumption.

## Preferred output-format default
A 12-week calendar (week-by-week table) showing what ships and what supporting promo runs alongside, a content backlog of 6 first-draft topics, a risk register, and a one-paragraph positioning statement. Flag any assumption about niche, platform, or cadence I should confirm before approving.
